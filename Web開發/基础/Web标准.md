主要包括`结构(Structure)`、`表现(Presentation)`、`行为(Behavior)`。

| 标准 | 说明                                                         |
| ---- | ------------------------------------------------------------ |
| 结构 | 对 **网页元素** 进行分类和整理。`HTML`文件作用的体现         |
| 表现 | 表现用于设置网页元素的版式、颜色、大小等 **外观样式**，主要指`CSS` |
| 行为 | 行为指网页模型的定义及 **交互** 的编写                       |
