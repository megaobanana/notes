| 物质的鉴定                                   | 物质的鉴别                     |
| -------------------------------------------- | ------------------------------ |
| ==确定==一种物质的成分或确定混合物中各种成分 | 将两种或两种以上的物质区别开来 |

离子检验的基本方法：`沉淀法(Cl-检验Ag)`、`显色法(SCN检验Fe)`、`气体法(HCl检验CO32-)`、或直接观察离子颜色


### 氯离子的检测

加入`硝酸银`，若加入稀硝酸后白色沉淀不溶解（消除`碳酸根`和`硫酸根`的影响），则说明有氯离子

| 离子        | 试剂或方法                                                   | 现象                                                         | 离子方程式                                                   |
| ----------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| OH^-^       | 石蕊试剂                                                     | 紫色变蓝色                                                   |                                                              |
| OH^-^       | 酚酞试剂                                                     | 变红色                                                       |                                                              |
| OH^-^       | 甲基橙试剂                                                   | 橙色变变黄                                                   |                                                              |
| H+          | 石蕊试剂                                                     | 紫色变红色                                                   |                                                              |
| H+          | 甲基橙试剂                                                   | 橙色变红色                                                   |                                                              |
| H+          | 碳酸钠 / 活泼金属                                            | 有气泡                                                       | Zn+2H^+^=Zn^2+^+H~2~↑$CO_3^{2-}+2H^+=H_2O+CO_2↑$             |
| S^2-^       | Pb(NO~3~)~2~溶液                                             | 产生黑色沉淀                                                 | Pb^2+^+S^2-^=PbS↓                                            |
| S^2-^       | CuSO~4~溶液                                                  | 产生黑色沉淀                                                 | Cu^2+^+S^2-^=CuS↓                                            |
| S^2-^       | 稀HCl或稀H~2~SO~4~                                           | 产生臭鸡蛋味                                                 | 2H^+^+S^2+^=H~2~S↑                                           |
| SO~4~^2+^   | 加入稀盐酸无沉淀生成也无气体放出，然后加入BaCl~2~。先用`盐酸酸化`意义：可以除去`亚硫酸根`（亚硫酸根加酸变二氧化硫和水），`碳酸根`（碳酸根加酸变二氧化碳和水），`银离子`（沉淀）的影响 | 产生白色沉淀                                                 | Ba^2+^+SO~4~^2-^=BaSO~4~↓                                    |
| SO~3~^2-^   | 加H^+^、产生的气体通入品红溶液（==不能使用硝酸酸化==）       | 有气体产生，能使`品红溶液褪色`（HSO~3~现象相同）             | $SO_3^{2-}+2H^+=H_2O+SO_2↑$                                  |
| $CO_3^{2-}$ | ==取上层清液==加H^+^、产生的气体通入澄清石灰水               | 有气体产生，`石灰水变浑浊`（HCO~3~现象相同）                 | $CO_3^{2-}+2H^+=H_2O+CO_2↑\\ Ca^{2+}+2OH^-+CO_2=CaCO_3↓+H_2O$ |
| Cl^-^       | Ag^+^，然后加入稀HNO~3~                                      | 产生白色沉淀，不溶于稀硝酸                                   | Ag^+^+Cl^-^=AgCl↓                                            |
| Br^-^       | Ag^+^，稀HNO~3~                                              | 产生<font color=#FFFDB9>淡黄色沉淀</font>，不溶于稀硝酸      | Ag^+^+Br^-^=AgBr                                             |
| Br^-^       | 苯或CCl~4~                                                   | 溶液分层，且苯层或CCl~4~层为<font color =#FF6726>橙红色</font> |                                                              |
| Br^-^       | 氯水                                                         | 溶液由淡黄绿色变黄色（水溶液中）或棕色（四氯化碳中）         | 2I^-^+Br~2~=I~2~+2Br^-^                                      |
| I^-^        | Ag^+^，然后加入稀HNO~3~                                      | 产生<font color=#FFD900>黄色沉淀</font>，不溶于稀硝酸        | Ag^+^+I^-^=AgI↓                                              |
| I^-^        | 新制氯水，淀粉溶液                                           | 溶液<font color=blue>变蓝</font>                             | 2I^-^+Cl~2~=I~2~+2Cl^-^，淀粉遇到I~2~<font color=blue>变蓝</font> |
| I^-^        | 苯或CCl~4~                                                   | 溶液分层，且苯层或CCl~4~层为<font color=#C00A85>紫红色</font> |                                                              |
| AlO^2-^     | 加HCl                                                        | 先生成白色沉淀，H^+^过量时沉淀溶解                           | $AlO_2^-+H^++H_2O=Al(OH)_3↓\\Al(OH)_3+3H^-=Al^{3+}+3H_2O$    |
| NH~4~^+^    | 加OH^-^，加热                                                | 有气体产生，气体使得湿润红色石蕊试纸变蓝；遇到蘸有HCl的玻璃棒有白烟 | 略                                                           |
| Ag^+^       | 加Cl^-^，稀硝酸                                              | 白色沉淀，不溶于稀硝酸                                       | 略                                                           |
| Mg^2+^      | 加OH^-^                                                      | 白色沉淀，碱过量沉淀不溶                                     | 略                                                           |
| Ba^2+^      | 加入稀硫酸                                                   | 白色沉淀                                                     | 略                                                           |
| Fe^2+^      | 加OH^-^                                                      | 白色沉淀->灰绿色沉淀->红褐色沉淀                             | 略                                                           |
| Fe^2+^      | 加硫氰化钾                                                   | 加KSCN无现象，再加入氯水溶液变红                             | 略                                                           |
| Fe^3+^      | 加硫氰化钾                                                   | 变红色                                                       |                                                              |
| Fe^3+^      | 加NaOH                                                       | 红褐色沉淀                                                   |                                                              |
| Fe^3+^      | 加苯酚C~6~H~6~OH                                             | 溶液显紫色                                                   |                                                              |
| Cu^2+^      | 加NaOH                                                       | 蓝色沉淀                                                     |                                                              |
| Al^3+^      | 加NaOH                                                       | 先生成白色沉淀，OH^-^过量后白色沉淀溶解                      |                                                              |

## 备注

如果检验前氯离子前加入了氢氧化钠，需要加入稀硝酸中和掉氢氧根，
