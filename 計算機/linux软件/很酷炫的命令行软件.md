## mapscii

```bash
sudo apt install nodejs npm
sudo npm install -g mapscii
```

或

```bash
telnet mapscii.me
```

然后使用mapscii启动软件

要滚动地图，请使用“向上”、“向下”、“向左”、“向右”箭头键。要放大/缩小位置，请使用 `a` 和 `z` 键。另外，你可以使用鼠标的滚轮进行放大或缩小。要退出地图，请按 `q`。
